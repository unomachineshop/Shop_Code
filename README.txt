To be used for generalized programming functionality. 
Add any code snippets for any language you might find,
feel free to upload them here. If there is not a 
folder for the language or device you are are working 
on, please add a folder, and push your code to that.

It is not limited purely to programming, for instance
in the raspberrypi/ folder, there is instructions
for setting up an LCD.
